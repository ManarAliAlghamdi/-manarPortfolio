Sand Sound

App Store

A sound-only mobile game where sound and haptics lead the experience — designed for visually impaired players but fun for everyone.

Overview

Sand Sound is a fully audio-based game built for blind and visually impaired players — but accessible and enjoyable by all. Players are placed in a pitch-dark desert environment, navigating lanes, avoiding obstacles, and solving challenges using only sound cues and vibrations. It redefines what a game can be when accessibility is the starting point, not an afterthought.

Features

  • 3-lane swipe navigation guided entirely by 3D positional sound
  • Real-time obstacle detection using audio cues and haptics
  • Immersive narrative tutorial that blends story and gameplay
  • Technologies: SwiftUI, AVFoundation, Core Haptics, MVVM architecture

My Role

As part of the development team, I focused on coding the core game mechanics — both in the immersive tutorial and the main gameplay. I implemented the 3-lane swipe navigation, obstacle interactions, and sound feedback system using SwiftUI and AVFoundation. To ensure a clean and scalable architecture, I structured my work using the MVVM pattern for smooth project organization and maintainability. Collaborating closely with the team, I helped ensure that the audio-first experience was intuitive, responsive, and fully accessible.
